# Core Faculty Model Evaluation (CFME) Project

This project is aiming to evaluate the Core Faculty Model implemented at Harvard
Medical School (HMS) starting 2025. Using secure AI models, the quality of
clinical clerkship evaluations will be checked against a rubric to test
differences before and after model implementation

## Data

Terminology:

- Evaluation: Anything using the work 'eval' in it relates to the original data
  of student evaluations by faculty during clerkship rotations
- Review: Anything that contains the word 'review' is related to the AI or human
  review and scoring of the provided evaluations to assess their quality

### Evaluations

The baseline data used are student evaluations written by clerkship faculty. The
file comes as one large spreadsheet and contains info about the students,
evaluators and clerkships. This data is parsed and inserted into an SQLite
database designed for this workflow.

### Reviews

Data here is generated by reviewing student evaluations using a rubric which has
been converted into a prompt to work for both AI and Humans, both who will score
each evaluation for quality.
